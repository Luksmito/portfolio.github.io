Pong

Jogo do pong implementado em arduino com uma tela OLED 128x64, fiz esse projeto para me desafiar a tirar proveito máximo do hardware, como desafio eu mesmo criei a biblioteca de controle da tela e de comunicação I2C, isso pode ser visto no meu github.

Osciloscópio

Se trata de um projeto de hardware e software, esse dispositivo é um osciloscópio, portanto mede um sinal de tensão ao longo do tempo, para fazer a medição, o hardware é um microcontrolador, nesse caso o arduino, onde utilizo o timer com intervalo de 40 microsegundos, essa medida pode ser alterada, no gif acima estou medindo a saída do arduino com uma frequência de aproximadamente 4khz.

CentralIOT

Esse projeto se trata de um projeto de hardware e software, o que foi feito aqui é um hardware que funciona como uma central de controle, basicamente você pode, com um protocolo estabelecido por mim, cadastrar dispositivos de IOT nessa central, dessa forma a central pode mandar comandos para esses dispositivos.

Sistema de agendamento

Um website feito com Django e Sqlite com o fim de servir como landing page e agendamentos, consiste de uma página principal de apresentação mais uma parte para agendamentos. Como esse é um paradigma comum em muitos empreendimentos fiz de forma genérica para servir para vários negócios, além disso foi feita uma interface mobile para verificar os agendamentos feitos e fazer outros.

CryptoID

App feito em Flutter, esse app utiliza a tecnologia de criptografia assimétrica para gerar e identificar assinaturas digitais, dessa forma é possível verificar fraudes de identidade ou validar informação de contratos.
